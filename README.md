## 项目介绍

本项目是一个记账 H5 页面前端项目（模仿微信记账本），基于Vue 3.0 + Vant 3.0 实现，主要包括明细、统计、个人中心、登录、注册、详情页等。

node v14.18.2

## 运行操作

安装依赖 npm install
运行项目 npm run serve
打包项目 npm run build
